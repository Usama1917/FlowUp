## Arabic / English / RTL

Arabic is the default language.

English must be supported.

When Arabic is active:

* layout must be RTL
* text alignment must be natural and clean
* Arabic text should not look cramped
* chat bubbles should align correctly
* room list should read correctly
* task details should align correctly
* forms and labels should be right/start aligned
* top navigation should still feel balanced

When English is active:

* layout switches to LTR
* text alignment switches correctly
* icons and directional arrows should behave naturally

Important mixed-direction rule:

Task codes, IDs, emails, URLs, file names, and technical values must always display as LTR islands even inside Arabic UI.

Examples:

* TASK-00045
* BIO-CH01-L02
* [flowup.manager@example.test](mailto:flowup.manager@example.test)
* design-final-v2.pdf

These should not break Arabic text direction.
