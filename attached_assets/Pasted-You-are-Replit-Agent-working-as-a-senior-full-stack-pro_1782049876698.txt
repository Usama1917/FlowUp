You are Replit Agent working as a senior full-stack product engineer and UI/UX engineer.

Build a complete modern web app prototype called **FlowUp**.

This prototype is for an internal company communication and task follow-up system.

Important: This is a standalone prototype first. Do not integrate with ERPNext yet. Build a polished, working demo that shows the full user experience clearly. Later, another developer will adapt it into ERPNext/Frappe.

## Main Goal

FlowUp is a modern chat-based task delivery and follow-up system for a company.

The company currently uses Telegram groups to send work between supervisors and designers. This causes confusion, lost files, unclear versions, and poor tracking.

FlowUp should replace that messy communication with a structured internal system:

* Departments
* Rooms/groups
* Chat
* Task cards
* File/submission flow
* Revision requests
* Approval
* Admin/governance
* Dashboard
* Audit log

The first MVP focuses on the **Design Department**, but the system should look expandable to other departments.

## Important Product Concept

Do not make this look like an ERP system.

Make it feel like a premium modern SaaS app:

* clean
* vibrant
* smooth
* Arabic-first
* RTL-correct
* mobile-friendly
* simple for non-technical employees

The designer user must feel like they are using a simple chat app, not a complex task management system.
